# Selectors

## Tag selector
`h1 {}`

## Class selector
`.classname`

## ID selector
`#idname`

## Direkte Nachfahren
`parent > children`

## Alle Nachfahren
`parent children`

## Alle Elemente
`*`

## Mehrere Selktoren
`p, h1, div`

## Benachbartes Geschwister-Element
`sibling1 + sibling2`

## Nachfolgende Geschwister
`sibling ~ following`

## Hintergrundbild
`background-image: url()`
